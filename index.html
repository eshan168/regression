Test for the Slope of a Regression Line

This slope t-test calculator performs a hypothesis test for the slope of a linear regression line. It calculates the regression equation, standard error of the slope, confidence interval, t-statistic, degrees of freedom, and p-value.

When to Use a Slope T-Test

A slope t-test is used to determine whether there is a statistically significant linear relationship between two quantitative variables. This test is commonly used in AP Statistics, introductory statistics courses, and college-level data analysis.

What This Calculator Shows

Least squares regression equation, Standard error of the slope, Confidence interval for the slope, T-test statistic and p-value, Regression graph
